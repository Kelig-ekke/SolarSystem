Задание: 
Разработать настольное приложение, которое визуализирует движение планет вокруг Солнца в двумерном пространстве с возможностью управления скоростью, масштабом и параметрами орбит.
Название: SolarSystemSimulator
Тип проекта: Desktop App
Технология:
— C# / .NET 8 (или 7)
— WPF (Canvas или DrawingVisual для отрисовки)
(альтернатива: Python + PyGame)

Функциональность (минимум)
— Базовый уровень (обязательно)
— На экране отображается Солнце и несколько планет (Меркурий, Венера, Земля, Марс и т.д.).
— Каждая планета вращается вокруг Солнца по круговой орбите.
— Скорость вращения зависит от радиуса орбиты (чем ближе к Солнцу — тем быстрее).
— Есть кнопки управления:
— — Старт / Пауза
— — Ускорить / Замедлить
— — Масштаб (+/−)

Средний уровень
— Отрисовка орбит (тонкие линии вокруг Солнца).
— Подписи к планетам (название, радиус орбиты).
— Центрирование камеры на выбранной планете (например, на Земле).
— Настройки через JSON (скорости, цвета, размеры планет).

Продвинутый уровень
— Эксцентрические орбиты (эллипсы).
— Наклон осей, вращение собственное.
— Спутники (например, Луна вокруг Земли).
— Расчёт позиций по формуле закона Кеплера: ω = sqrt(G * M / r³)
где G — гравитационная постоянная, M — масса Солнца, r — расстояние до центра.
—Визуализация траекторий при ускорении времени.
— Темная / светлая тема.
