# Solar System Simulator

**Симулятор Солнечной системы**

Учебный проект студентов СПбГЛТУ им. С.М. Кирова 2 курса группы ЛПб-ИСиТ-24-1 

Лебедевой Е.Н. и Гавриловой А.Р.

## Установка
### Требования
Python 3.8 или выше
Pygame 2.5+

## Управление
### Клавиатура

Кнопка          | Действие
----------------|----------------------
Стрелка вверх   | Увеличить скорость
Стрелка вниз    | Уменьшить скорость
"+"             | Увеличить масштаб
"-"             | Логика симуляции
R               | Сброс вида
Пробел          | Пауза/продолжить


### Мышь
Кнопка          | Действие
----------------|----------------------
Колесо мыши  | Изменить масштаб
ЛКМ | Нажать на кнопки интерфейса


### Скорость симуляции
Фиксированные значения: 0.1x, 0.5x, 1x, 2x, 5x, 10x, 20x, 30x, 40x, 50x, 80x, 100x

### Масштаб
Фиксированные значения: 0.01x, 0.05x, 0.1x, 0.5x, 1x, 2x, 5x, 8x

### Настройка
Параметры планет можно изменить в файле settings.json

По умолчанию заданы реальные параметры планет, но пропорции (соотношение диаметра планет к расстоянию между ними) искажены для наглядности.

## Структура проекта

Название файла  | Содержание файла
----------------|----------------------
main.py         | Основной файл с циклом игры
simulation.py   | Логика симуляции
planet.py       | Класс планеты
settings.json   | Параметры планет
